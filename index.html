Table Making Rules
Tag Name Meaning
<table> Table Create a table
<caption> Caption Add a caption
<th> Table Header Defines header cell
<tr> Table Row Creates row
<td> Table data Creates empty cell
<thead> Table Head Encloses header elements
<tbody> Table Body Encloses body elements
<tfoot> Table Footer Encloses footer content
<col> Column Specifies column elements
<colgroup> Column Group Encloses <col> elements
